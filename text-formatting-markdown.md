# Formatação de texto em Markdown
- Aqui estão as principais opções de formatação de texto em Markdown:

    - **Negrito**: `**texto**` ou `__texto__`
    - *Itálico*: `*texto*` ou `_texto_`
    - ***Negrito e itálico***: `***texto***`
    - `Código`: `` `código` ``
    - ~~Tachado~~: `~~texto~~`
    - > Citação: `> texto`
    - [Link](https://example.com): `[texto](url)`
    - Sobrescrito: `texto^sobrescrito^`
    - Subscrito: `texto~subscrito~`
    - Título 1: `# Título`
    - Título 2: `## Título`
    - Listas numeradas: `1. item`, `2. item`
    - Listas com marcadores: `- item` ou `* item`
    - Separador horizontal: `---` ou `***`
    - Tabelas: Use `|` para delimitadores de coluna
    - Imagens: `![texto alternativo](url-da-imagem)`

    **Nota:** Markdown padrão não suporta sublinhado. Use HTML: `<u>texto sublinhado</u>`
